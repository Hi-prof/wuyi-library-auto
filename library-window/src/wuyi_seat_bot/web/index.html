账号管理

服务端配置

填好「服务端 URL」，Bearer Token 也可以先留空，后面再补。

URL 怎么填
  • 填到协议 + 域名（或 IP）+ 端口为止，例如 https://srv.example.com 或 http://192.168.1.10:5000。
  • 不要带 /api/v1、/accounts 之类的路径，也不要带 ?key=value 查询串；客户端会自己拼。
  • 末尾的 / 会被自动剥掉，https://srv.example.com/ 与不带斜杠等价。
  • 生产部署强烈建议使用 https://；服务端开启了 426 校验时只能填 https://。
  • 本机调试可以填 http://127.0.0.1:5000，并把下方「校验 TLS 证书」取消勾选。

两项都留空时点击「保存」会清除当前服务端配置，回到「未配置服务端」状态，本地任务执行不受影响。
