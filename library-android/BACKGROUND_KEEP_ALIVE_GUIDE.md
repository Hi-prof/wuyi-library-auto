# 后台保活配置指南

为确保图书馆自动预约应用能够正常运行并及时签到，请按照以下步骤配置手机的后台保活设置。

## 一、通用设置（适用于所有 Android 手机）

### 1. 关闭电池优化
1. 打开手机的 **设置** → **电池** → **电池优化**
2. 找到 "武夷图书馆自动预约" 应用
3. 选择 **不优化** 或 **允许后台运行**

### 2. 允许自启动
1. 打开手机的 **设置** → **应用管理** → **自启动管理**
2. 找到 "武夷图书馆自动预约" 应用
3. 开启 **允许自启动**

### 3. 锁定后台
1. 打开最近任务（多任务）界面
2. 找到 "武夷图书馆自动预约" 应用
3. 下拉应用卡片或点击锁定图标，防止被系统清理

### 4. 允许后台网络访问
1. 打开手机的 **设置** → **WLAN** → **更多设置**
2. 确保 **休眠时保持 WLAN 连接** 设置为 **始终**

## 二、品牌手机特定设置

### 小米 / Redmi / POCO
1. **省电优化**：设置 → 电池与性能 → 应用电量管理 → 武夷图书馆自动预约 → 设置为 **无限制**
2. **自启动**：设置 → 应用设置 → 授权管理 → 自启动管理 → 开启
3. **后台弹出界面**：设置 → 应用设置 → 授权管理 → 后台弹出界面 → 允许
4. **锁定后台**：在最近任务中下拉应用卡片并点击锁定图标
5. **关闭 MIUI 优化**：设置 → 更多设置 → 开发者选项 → 关闭 MIUI 优化（可选）

### 华为 / 荣耀
1. **应用启动管理**：设置 → 应用 → 应用启动管理 → 找到应用并设置为 **手动管理**，全部开启
2. **忽略电池优化**：设置 → 电池 → 更多电池设置 → 休眠时始终保持网络连接
3. **锁定后台**：在最近任务中下拉应用卡片并点击锁定图标

### OPPO / OnePlus / Realme
1. **电池优化**：设置 → 电池 → 高耗电应用 → 武夷图书馆自动预约 → 允许后台运行
2. **自启动**：设置 → 应用管理 → 自启动 → 开启
3. **后台冻结**：设置 → 电池 → 应用耗电管理 → 智能后台冻结 → 关闭（或将应用加入白名单）
4. **锁定后台**：在最近任务中点击应用右上角的锁定图标

### vivo / iQOO
1. **后台高耗电**：i 管家 → 应用管理 → 权限管理 → 后台高耗电 → 允许
2. **自启动**：i 管家 → 应用管理 → 自启动 → 开启
3. **电池优化**：设置 → 电池 → 后台耗电管理 → 不限制
4. **锁定后台**：在最近任务中点击应用上方的锁定图标

### 三星
1. **电池优化**：设置 → 应用程序 → 武夷图书馆自动预约 → 电池 → 后台使用限制 → 不限制
2. **休眠应用**：设置 → 电池和设备维护 → 电池 → 后台使用限制 → 从不休眠的应用 → 添加应用
3. **自动优化**：设置 → 电池和设备维护 → 关闭自动优化

## 三、权限检查清单

请确保应用已获得以下权限：

- ✅ **通知权限**：用于显示签到状态和守护服务通知
- ✅ **网络权限**：用于连接图书馆系统
- ✅ **蓝牙权限**：用于定位是否在图书馆内（如果使用蓝牙签到功能）
- ✅ **定位权限**：用于蓝牙扫描（Android 要求）
- ✅ **前台服务权限**：用于保持后台运行
- ✅ **精准闹钟权限**：用于准时触发签到任务

## 四、验证后台保活是否成功

1. **查看通知栏**：应该能看到 "守护调度服务" 的常驻通知，显示当前活动任务数
2. **查看应用内状态**：打开应用，检查守护服务是否正常运行
3. **锁屏测试**：锁屏 10-15 分钟后解锁，查看应用是否仍在后台运行
4. **查看系统日志**：在应用内查看运行日志，确认没有 "进程被系统回收" 的警告

## 五、常见问题

### 1. 应用经常被杀后台怎么办？
- 检查是否完成了上述所有设置
- 尝试卸载手机管家类第三方应用（如 360、腾讯管家等）
- 在开发者选项中关闭 "不保留活动" 和 "后台进程限制"

### 2. 签到经常超时失败怎么办？
- 确保在签到时间段（通常是早上 7:30-8:30）手机联网且屏幕解锁至少一次
- 检查是否连接到校园网 Wi-Fi
- 在应用内手动执行一次签到测试

### 3. 收不到通知怎么办？
- 检查系统设置中是否允许应用通知
- 检查通知权重设置，建议设置为 "重要" 或 "紧急"
- 确认手机的勿扰模式设置没有屏蔽应用通知

### 4. 蓝牙扫描失败怎么办？
- 确保蓝牙已开启
- 检查是否授予了定位权限（Android 要求）
- 尝试在图书馆内手动扫描测试

## 六、开发者选项设置（可选，仅限高级用户）

如果你熟悉 Android 开发者选项，可以进行以下设置：

1. **后台进程限制**：设置 → 系统 → 开发者选项 → 后台进程限制 → 标准限制
2. **不保留活动**：设置 → 系统 → 开发者选项 → 不保留活动 → **关闭**
3. **后台检查**：设置 → 系统 → 开发者选项 → 后台检查 → **关闭**

---

**注意**：不同品牌、不同系统版本的设置界面可能略有差异，请根据实际情况调整。

如果按照以上步骤设置后仍有问题，请查看应用内的运行日志并联系开发者。
